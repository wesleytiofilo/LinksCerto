Model (modelo) no MVC

O model representa a camada de dados da aplicação. Ele gerencia a lógica de negócios, as regras e a interação com o banco de dados.

Principais Funções do Model

  • Armazenar e gerencia dados
  • Executa regras de negócio
  • Interage com o banco de dados

Exemplo

Imagine um sistema onde o Model contem as informaçoes como o nome,e-mail e senha, além da regras de validação.
