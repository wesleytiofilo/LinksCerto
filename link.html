Introdução ao MVC

O padrão MVC (Model-View-Controller) é uma arquetetura de software ultilizada no desenvolvimento de aplicações para separar a lógica de negócios, interface do usuário e o controle dos dados.

Componetes do MVC

  • Model(modelo) - Representa os dados e a lógica de negócios da aplicação
  • View (visão) - Responsável pela interface com o usuário e exibição de dados
  • Controller (Controlador) - Intermedia a comunicação entre Model e View
